# <img src="over.png" width="32" height="32" align="center" style="border-radius: 8px;"> OverTube

> **Take YouTube Beyond Its Limits.** A modern, performance-oriented Chrome extension that enhances and personalizes your YouTube viewing experience with modular settings and a dark-red premium dashboard.

---

### Core Features

| Feature | Description |
| :--- | :--- |
| **Custom Progress Bar** | Swap YouTube's default progress bar with presets (Neon, Minimal, Gradient, Gaming) and customized character GIF scrubbers. |
| **Enhanced Playback Speed** | Unlocks speed multipliers up to 3.0x with persistent settings application. |
| **Audio Booster** | Safe Web Audio API amplification ranging from 0% to 500% with built-in clipping limiters. |
| **Focus Mode** | Clean, modular toggles to hide comments, sidebar feeds, home page shelves, and end screens. |
| **Hide Shorts** | Removes Shorts guides, shelves, and search cards completely across the platform. |
| **Sticky Mini Player** | Smoothly floats the active video into a draggable, interactive overlay upon scrolling down. |

---

> [!IMPORTANT]
> **Installation & Usage**
> 1. Load the extension unpacked in `chrome://extensions` in Google Chrome.
> 2. Open or refresh your YouTube tabs to initialize.
